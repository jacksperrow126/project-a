# Valy Life - Flutter App

A cross-platform mobile application built with Flutter for iOS and Android.

## Getting Started

### Prerequisites
- Flutter SDK (>=3.0.0)
- Dart SDK
- Xcode (for iOS development)
- Android Studio (for Android development)

### Installation

1. Install dependencies:
```bash
flutter pub get
```

2. Run the app:
```bash
flutter run
```

### Building

#### iOS
```bash
flutter build ios
```

#### Android
```bash
flutter build apk
# or
flutter build appbundle
```

## Project Structure

```
lib/
  main.dart          # Entry point
  models/            # Data models
  screens/           # UI screens
  services/          # API services
  widgets/           # Reusable widgets
```
